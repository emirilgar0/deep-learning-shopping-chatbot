Deep Learning Based Shopping Chatbot
Bu proje, müşteri alışveriş verilerini analiz ederek bir derin öğrenme modeli eğiten ve bu modeli web tabanlı bir chatbot arayüzü üzerinden sunan uçtan uca bir uygulamadır.

 Özellikler
Model Eğitimi: TensorFlow kullanılarak müşteri davranışları üzerine eğitilmiş derin öğrenme modeli.

Chatbot Arayüzü: Kullanıcıların doğal dilde sorular sorarak tahminler alabildiği interaktif web arayüzü.

Full-Stack Entegrasyon: Python (AI) ve Node.js (Backend) teknolojilerinin birlikte kullanımı.

 Teknolojiler
Yapay Zeka: Python, TensorFlow, Pandas, Scikit-learn

Backend: Node.js, Express.js

Frontend: HTML, CSS, JavaScript
